# 数感 v3

**数字の勘を、1分ずつ鍛える。** 8問・最大60秒で終わる、サーバー不要の静かな数字ゲームです。

## v3で入れたこと

- 正解ポイント、速さボーナス、コンボボーナスを分けたスコア
- カテゴリごとの目標時間。速さは正解した場合だけ加点
- 自己ベスト、最速パーフェクト、最大コンボ、レベル到達日だけを保存
- 通常プレイの低い点数・誤答・未実施日・連続日数は保存しない
- 誤答タグを、その回だけの「今日の攻略メモ」に変換
- 18テンプレートを数値バリエーション化。答えの丸暗記を防止
- 結果画面は「記録 → 攻略メモ → 間違えた問題の解説 → 全問解説」の順
- 現実数字を出題した回だけ、基準日と出典付きの「今日知った数字」を表示

## GitHub Pagesで公開する

1. このZIPを解凍する。
2. 中身をGitHubリポジトリのルートに置く。
3. GitHubの **Settings → Pages** を開く。
4. Sourceを **Deploy from a branch**、Branchを `main`、Folderを `/ (root)` にして保存する。
5. 表示されたURLを開く。

ビルド、サーバー、依存パッケージは不要です。

## ローカルで試す

`index.html` をブラウザで開くだけで動きます。ブラウザのプライベートモードやLocalStorageを無効にした環境では、プレイはできますが記録帳は残りません。

## 保存データ

`LocalStorage` には、自己ベスト、最速パーフェクト、最大コンボ、レベル到達日時、累計正解数、直近出題テンプレートだけを保存します。通常プレイの失敗履歴・誤答ログは保存しません。

## アイコン

`icons/` に、favicon、Apple touch icon、PWA / Android向けPNG、maskable icon、OG画像、マスターWebPを用意しています。
