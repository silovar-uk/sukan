# 数感 v3.2

**数字の勘を、1分ずつ鍛える。**

8問・最大60秒で終わる、サーバー不要の静かな数字ゲームです。GitHub Pagesにそのまま公開できます。

## v3.2で追加したこと

- 「だいたいどれくらい？」を問う **現実の数字100問** を追加
- ジャンルは、日本・暮らし／世界・人口／地球・地理／宇宙／人体・生命／動物・自然／時間・テクノロジー
- 1プレイに「現実の数字」を必ず2問出題
- 問題タグにジャンルを表示し、何のスケール感を扱っているか分かるように変更
- 各問題に、答え・比較の目安・基準日または出典リンクを内蔵
- 直近に出た問題は避け、同じ現実数字が続きにくいように調整

## 数感の主な仕様

- 4画面：ホーム → カウントダウン → ゲーム → 結果
- 正解ポイント、速さボーナス、コンボボーナスを分けたスコア
- カテゴリごとの目標時間。速さは正解した場合だけ加点
- 自己ベスト、最速パーフェクト、最大コンボ、レベル到達日だけをLocalStorageに保存
- 通常プレイの低い点数・誤答・未実施日・連続日数は保存しない
- 時間切れで未回答になった問題も、結果画面で答え・解説・コツを確認可能
- 誤答タグは、その回だけの「今日の攻略メモ」に変換

## GitHub Pagesで公開する

1. このZIPを解凍する。
2. 中身をGitHubリポジトリのルートに置く。
3. GitHubの **Settings → Pages** を開く。
4. Sourceを **Deploy from a branch**、Branchを `main`、Folderを `/ (root)` にして保存する。
5. 表示されたURLを開く。

ビルド、サーバー、依存パッケージは不要です。

## データについて

現実の数字は「桁感をつかむ」ための概数です。人口・利用者数・施設数など変動しうる数値には、アプリ内で基準日・出典リンクを表示します。公開後も、定期的に数字とリンクを見直してください。
